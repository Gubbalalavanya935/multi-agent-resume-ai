# 🧠 Agentic AI Resume Analyzer (LLM + Multi-Agent System)

🚀 AI-powered system that analyzes resumes and matches them with job roles using **LLMs + Multi-Agent architecture**

---

## 🔥 Key Features

* 🤖 Multi-agent system using **ReAct framework**
* 📄 Resume parsing using AI
* 🎯 Intelligent job matching
* ⚡ FastAPI backend with optimized response time
* 🌐 Deployed live application

---

## 🛠️ Tech Stack

* Python
* FastAPI
* LLM (Groq API)
* Streamlit
* PyPDF
* JSON
* Render (Deployment)

---
